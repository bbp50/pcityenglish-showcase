How these quizzes are built

From real mistakes to targeted practice

Between lessons, I keep a running log of each student's recurring errors — drawn directly from corrections made during live sessions and notes taken throughout our work together. Over time, these logs reveal patterns: the same article dropped in the same construction, the same preposition swapped, the same tense used where another is needed.

Those patterns become the quiz. Each set of questions targets exactly the errors that came up most for that student — not a generic grammar textbook, but a precise record of what actually needs work. The result is a practice tool that feels specific because it is.

Every question includes instant feedback that names the rule and explains the reasoning, so students leave with a clearer mental model — not just a score.
